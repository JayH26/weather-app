# 🌤 Weather App

Aplicación web desarrollada en React que consume la API de [Open-Meteo](https://open-meteo.com/) para mostrar información meteorológica en tiempo real de cualquier ciudad del mundo.

## 🚀 Funcionalidades

- Búsqueda de clima por nombre de ciudad
- Geolocalización automática del navegador
- Vista de clima actual con temperatura, viento, humedad y condición meteorológica
- Pronóstico de los próximos 7 días con temperaturas máximas, mínimas y precipitación
- Manejo de estados de carga y errores
- Diseño responsivo y moderno

## 🛠 Tecnologías utilizadas

- React 18
- JavaScript (ES6+)
- CSS3 (Flexbox, Grid, Backdrop Filter)
- Open-Meteo API (clima y geocoding)
- Git / GitHub

## 📦 Instalación y uso

1. Clona el repositorio:
```bash
   git clone https://github.com/JayH26/weather-app.git
```

2. Entra a la carpeta:
```bash
   cd weather-app
```

3. Instala las dependencias:
```bash
   npm install
```

4. Inicia la aplicación:
```bash
   npm start
```

5. Abre tu navegador en `http://localhost:3000`

## 🌐 APIs utilizadas

- **Clima:** `https://api.open-meteo.com/v1/forecast`
- **Geocoding:** `https://geocoding-api.open-meteo.com/v1/search`
